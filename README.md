# JSMedium
A Full-Stack Blog site.

### Tech Stack:
- Nodejs
- Express
- Mongoose
- MongoDB

### Working:
- Clone the repo to your machine
- Install all dependencies using ``` npm i ```
- Fire up MongoDB using ``` mongod ```

- To run the server do:
```
nodemon script.js
```

- Open your browser and go to ``` localhost:3000 ```
